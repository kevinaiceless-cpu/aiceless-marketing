# Aiceless Marketing Site

Professional marketing website for Aiceless business alerts platform.

## Features

- ✅ Landing page with hero, features, and CTAs
- ✅ Mission statement page
- ✅ Pricing page (3 tiers: Essential, Professional, Enterprise)
- ✅ Privacy Policy (Facebook/Instagram compliant)
- ✅ Terms of Service
- ✅ About Us page
- ✅ Responsive design
- ✅ Aiceless brand colors

## Quick Start

### Local Development

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

### Deploy to Vercel

1. Push this code to GitHub
2. Import project in Vercel
3. Add your custom domain: `aiceless.com`
4. Deploy!

Or use Vercel CLI:

```bash
npm install -g vercel
vercel
```

## Domain Setup

### In Vercel:
1. Go to Project Settings → Domains
2. Add `aiceless.com` and `www.aiceless.com`
3. Follow DNS instructions

### In Namecheap:
1. Advanced DNS tab
2. Add A Record: `@` → `76.76.21.21`
3. Add CNAME: `www` → `cname.vercel-dns.com`
